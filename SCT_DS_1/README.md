# SkillCraft Technology Data Science Internship

## Task 1: Population Distribution Visualization

### Objective

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Dataset

World Bank Population Dataset (SP.POP.TOTL)

### Tools Used

* Python
* Pandas
* Matplotlib

### Visualizations

1. Bar Chart: Top 10 Most Populous Countries (2024)
2. Histogram: Distribution of Population Across Countries (2024)

### Project Workflow

1. Downloaded population dataset from the World Bank.
2. Cleaned and prepared the data using Pandas.
3. Created visualizations using Matplotlib.
4. Saved charts as PNG files.

### Results

The visualizations provide insights into population distribution across countries and highlight the most populous nations in the world.
