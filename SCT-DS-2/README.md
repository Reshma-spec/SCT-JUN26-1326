# SkillCraft Technology Data Science Internship - Task 2

## Objective

Perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic dataset.

## Dataset

Titanic Dataset from Kaggle

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Data Cleaning

* Handled missing values
* Removed unnecessary columns
* Prepared data for analysis

## Exploratory Data Analysis

* Age Distribution
* Survival by Gender
* Survival by Passenger Class
* Correlation Analysis

## Key Findings

* Females had a higher survival rate than males.
* First-class passengers had better survival chances.
* Age showed variation across passengers.
* Passenger class had a significant relationship with survival.
