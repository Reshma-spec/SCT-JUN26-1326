# Customer Purchase Prediction Using Decision Tree Classifier

This project was completed as part of the SkillCraft Technology Data Science Internship (Task 3).

## Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

## Dataset

Bank Marketing Dataset from the UCI Machine Learning Repository.

## Tools and Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Project Workflow

1. Loaded and explored the Bank Marketing dataset.
2. Performed data preprocessing and label encoding for categorical features.
3. Split the dataset into training and testing sets.
4. Built a Decision Tree Classifier model.
5. Evaluated the model using accuracy and confusion matrix.
6. Visualized the decision tree structure.

## Key Findings

* Customer demographics and previous marketing interactions influence subscription decisions.
* The Decision Tree model effectively classifies whether a customer will subscribe to a term deposit.
* Feature-based splitting helps identify important decision factors.

## Outcome

Successfully developed a machine learning classification model capable of predicting customer purchase behavior and gained practical experience in data preprocessing, model training, evaluation, and visualization.
